# Лабораторні роботи з курсу &laquo;Безпека програм та даних&raquo;

Цей репозиторій містить вихідний код та інструкції запуску програм до лабораторних робіт з навчальної дисципліни &laquo;Безпека програм та даних&raquo;.

## Структура репозиторію

Репозиторій структуровано згідно з вимогами - кожна лабораторна робота знаходиться в окремій директорії.

| Директорія | Назва лабораторної роботи             |
| :--------- | :------------------------------------ |
| `lab01`    | Аудит власної цифрової безпеки        |
| `lab02`    | Захист особистих повідомлень          |
| `lab03`    | Невидимі дані у власних файлах        |
| `lab04`    | Особистий цифровий підпис             |
| `lab05`    | Захищена електронна пошта             |
| `lab06`    | Етичний хакінг власного застосунку    |
| `lab07`    | Комплексний захист особистого проекту |

## Інструкції з використання

Всі програми в репозиторії написані на Python, тому вимагають наявності встановленого Python 3. Якщо Python не встановлений, його можна завантажити з офіційного сайту: https://www.python.org/downloads/.

Інструкції з запуску кожної окремої програми містяться у `README.md` в директорії відповідної програми.
